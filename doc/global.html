gameBoard

The game board is a 2D array, containing either Piece objects or 0. N.B.: JavaScript, being JavaScript, has no reliable way to compare inner arrays in 2D arrays. The hack used throughout is to iterate over the outer array, then Array.toString() the inner arrays to compare with other Array.toString()'ed functions. This file is included in the HTML document, at the top. Thus, it defines gloval variables for the project.
